# 파이썬 수업자료 및 예제파일 저장소

>파이썬 기초 강의 수업 자료는 [학원 홈페이지](https://www.firstcoding.kr/) [프로그램] → [강의자료] → [파이썬 강의자료 PDF]를 참고하시기 바랍니다.

## 주피터 노트북(Jupyter Notebook) 설치
- pip 업그레이드하기
  ```
  pip install --upgrade pip
  ```
- 주피터 노트북(Jupyter Notebook) 패키지 다운받기
  ```
  pip install jupyter
  ```
- CMD창에서 다음 명령으로 실행한다.
  ```
  jupyter notebook
  ```
